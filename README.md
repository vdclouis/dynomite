# Dynomite
### The interactive climbing app

## Deploy
1. run `npm install`
2. run `bower install`
3. run `grunt server`

## Technologies
### MEAN stack
* __M__ongoDB
* __E__xpressJS
* __A__ngularJS
* __N__odeJS
