# Dynomite
The interactive climbing app

## Prerequisites
1. Node & NPM
2. Bower
3. Mongo
4. Git

## Deploy
1. run `mongod`
2. run `npm install`
3. run `bower install`
4. run `npm start`

## Technologies
### MEAN stack
* __M__ongoDB
* __E__xpressJS
* __A__ngularJS
* __N__odeJS
